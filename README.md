# Token Gated Captive Portal
This Project attempts to provide a blockchain alt solution to granting/leasing access to wifi using Unlock Protocol for basic free daily subscriptions, Openwisp & Raduis to create & authenticate users to use session info provided from onchain sources. Privy provides simple email & Biconomy assist us with funding gas for user subscriptions


## Getting Started

First, run:
```bash
npm i 
# or
npm install
```

create .env.local in root & add values matching `environment.d.ts`

```bash
npm run dev
```
